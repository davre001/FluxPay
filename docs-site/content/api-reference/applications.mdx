# Applications

All endpoints require authentication.

| Method   | Path                                | Query      |
| -------- | ----------------------------------- | ---------- |
| `GET`    | `/api/applications/mine`            | `?status`  |
| `GET`    | `/api/applications/incoming`        | —          |
| `POST`   | `/api/applications/:id/withdraw`    | —          |
| `POST`   | `/api/applications/:id/reject`      | —          |

* **`mine`** — creator's sent applications
* **`incoming`** — brand inbox: applications across the org's own jobs
* **`withdraw`** — creator withdraws their pending application
* **`reject`** — brand declines a pending applicant
